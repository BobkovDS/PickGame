HOW TO BUILD:
	1) Copy build.bat to parent dir for source dir
	2) Be sure that you have FBX SDK in <parent_for_source_dir>/../../../FBXSDK dir
	3) Run build.bat
	4) Open single PickGame project (not full solution) and configure WinPixEventRuntime package in NuGet Package Manager
